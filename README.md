## Setup
### 1. Requirements
 + PHP
 + MariaDB
 + Composer
 + Symfony

### 2. Clone the repo
```bash
git clone https://github.com/rekky1aws/doit.git
```

### 3. Install dependencies
```bash
composer install
```

### 4. Run
```bash
symfony server:start
```
